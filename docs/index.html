Что внутри

Один exe, без установщика, без рекламы

🎴

Мини-плеер оверлей

Компактный виджет с обложкой, названием трека и управлением — можно закрепить в любом углу экрана.

⌨

Горячие клавиши

Play/Pause, следующий трек, лайк/дизлайк — настраиваются под себя и работают даже без фокуса окна.

❤

Лайк/дизлайк в фоне

Клик отправляется напрямую в страницу через DevTools Protocol — переключаться на Яндекс Музыку не нужно.

↻

Автообновление

Проверяет новые версии на GitHub и предупреждает перед установкой — без скачивания вручную.

📦

Один файл

Никаких лишних DLL рядом — всё нужное упаковано прямо в YMHub.exe.

🛠

Диагностика

Логи работы видны прямо в настройках Яндекс Музыки — без отдельных программ и консолей.

Как начать

1
Скачайте YMHub.exeОдин файл, ничего распаковывать не нужно.

Запустите рядом с Яндекс МузыкойYMHub сам подключится и настроит всё необходимое.
